# ESP32 PIN

| esp32 | hoverboard | PIN |
| --- | --- | --- |
| 3v3 |  SWD | 3v3 |
| RX 16 | Left Side | PA2 TX Blue |
| TX 17 | Left Side | PA3 RX Green |
| GND | Left Side | GND Black |
